## ―ニューラルネットワークによる擬似乱数列の予測可能性評価―

### 1．企画概要

本企画では、異なる方法で生成されたビット列に対して、過去のビット列から次の1ビットを予測するニューラルネットワークを作成する。

線形帰還シフトレジスタ（LFSR）が生成するビット列には線形な規則性があるため、十分な出力を観測すれば将来の値を予測できる。一方、暗号学的用途で使用される乱数生成器には、出力から将来の値を予測することが困難であるという性質が求められる。

そこで、LFSR、非線形処理を加えたシフトレジスタ、暗号学的乱数生成器から生成したビット列をニューラルネットワークに学習させ、生成方法によって次ビット予測精度がどのように変化するかを比較する。

### 2．目的

本企画の目的は、ニューラルネットワークがビット列に含まれる規則性をどの程度学習できるかを調べることである。

特に、次の点を検証する。

1. LFSRが生成したビット列をニューラルネットワークで予測できるか。
2. 線形な生成器に非線形処理を加えることで予測精度が変化するか。
3. 暗号学的乱数生成器の出力に対する予測精度が50％付近になるか。
4. ニューラルネットワークに与える過去のビット数が予測性能に与える影響。
5. ニューラルネットワークによる予測と、LFSR専用の代数的復元手法との違い。

### 3．対象とする乱数生成器

次の3種類の生成器を比較する。

#### 3.1 線形帰還シフトレジスタ

12ビットのLFSRを使用する。

レジスタ中の複数のビットの排他的論理和によって、次の状態を決定する。最大周期となるような帰還多項式を選んだ場合、周期は

[
2^{12}-1=4095
]

となる。

#### 3.2 非線形フィルタ付きLFSR

LFSRの状態更新は線形のままとし、出力に非線形関数を適用する。

例えば、内部状態のビットを用いて、

[
z_t
===

s_{t,0}
\oplus
(s_{t,3}\land s_{t,7})
\oplus
s_{t,11}
]

のように出力を生成する。

これにより、単純なLFSRと比較して、非線形処理がニューラルネットワークの予測精度に与える影響を調べる。

#### 3.3 暗号学的乱数生成器

Pythonの`secrets`モジュールを使用してビット列を生成する。

このビット列を、ニューラルネットワークで容易に学習できる規則性を持たないと考えられる比較対象として使用する。

### 4．データセットの作成

各生成器から約4,000ビットの系列を生成する。

生成されたビット列を

[
z_0,z_1,\ldots,z_{N-1}
]

とする。過去の (k) ビットを入力とし、その直後の1ビットを教師ラベルとする。

[
x_t=(z_t,z_{t+1},\ldots,z_{t+k-1})
]

[
y_t=z_{t+k}
]

例えば (k=12) の場合、過去12ビットから13番目のビットを予測する二値分類問題となる。

入力長は、主に次の2種類を比較する。

[
k=6,\ 12
]

12ビットLFSRに対し、内部状態長より短い入力と、内部状態長と同じ長さの入力で予測性能が変化するかを確認する。

### 5．データの分割

時系列データでは、隣接する入力データの大部分が重複する。そのため、作成した入力データを無作為に分割すると、非常によく似たデータが学習用とテスト用の両方に含まれる可能性がある。

これを避けるため、元のビット列を時系列順に次のように分割する。

* 学習用：全体の約60％
* 検証用：全体の約20％
* テスト用：全体の約20％

系列を分割した後、それぞれの区間から入力データと教師ラベルを作成する。

### 6．ニューラルネットワーク

TensorFlowおよびKerasを用いて、小規模な多層パーセプトロンを実装する。

ネットワーク構造は次の通りとする。

* 入力層：過去の (k) ビット
* 全結合層：32ユニット
* ReLU活性化関数
* 全結合層：16ユニット
* ReLU活性化関数
* 出力層：1ユニット

出力は、次のビットが1である確率として扱う。

学習条件は次のように設定する。

| 項目      | 設定               |
| ------- | ---------------- |
| フレームワーク | TensorFlow・Keras |
| 損失関数    | 二値交差エントロピー       |
| 最適化手法   | Adam             |
| 学習率     | 0.001            |
| バッチサイズ  | 32               |
| 最大エポック数 | 50               |
| 過学習対策   | Early Stopping   |

モデルの規模は約1,000パラメータ以下であり、GPUを使用せず、ノートPCのCPUで学習可能な大きさとする。

### 7．実験内容

#### 実験1：乱数生成方式による比較

同一構造のニューラルネットワークを使用して、次のビット列をそれぞれ学習する。

* LFSR
* 非線形フィルタ付きLFSR
* `secrets`によるビット列

各生成器に対するテストデータの予測精度を比較する。

#### 実験2：入力長による比較

入力長を

[
k=6,\ 12
]

に変更し、ニューラルネットワークが参照できる過去のビット数によって予測精度が変化するかを調べる。

#### 実験3：学習データ数による比較

学習に使用するサンプル数を変更し、LFSRの規則を学習するために必要なデータ量を調べる。

#### 実験4：代数的手法との比較

LFSRに対してBerlekamp–Massey法を適用する。

ニューラルネットワークによる学習と、LFSRの線形構造を直接利用する代数的手法について、次の点を比較する。

* 必要な観測ビット数
* 予測精度
* 計算時間
* 異なる初期状態に対する予測性能

### 8．評価方法

主な評価指標として、次の値を使用する。

* 学習データに対する予測精度
* 検証データに対する予測精度
* テストデータに対する予測精度
* 二値交差エントロピー損失
* 学習時間
* 学習損失の推移

乱数列の0と1がほぼ均等である場合、規則性を学習できていないモデルの予測精度は約50％になると考えられる。

ニューラルネットワークの初期値による結果の変動を確認するため、各条件について3回学習を行い、予測精度の平均値と標準偏差を求める。

### 9．予想される結果

LFSRでは、入力長と学習データ数が十分であれば、ニューラルネットワークがビット列中の線形な規則性を学習し、50％を上回る予測精度を示すと予想される。

非線形フィルタ付きLFSRでは、単純なLFSRよりも予測精度が低下すると考えられる。ただし、非線形関数が単純である場合には、ニューラルネットワークがその規則を学習できる可能性がある。

`secrets`から生成したビット列については、テストデータに対する予測精度が50％付近になると予想される。

また、LFSRに対しては、ニューラルネットワークよりも線形構造を直接利用するBerlekamp–Massey法の方が、少ないデータで正確に生成規則を復元できると考えられる。

### 10．考察上の注意

ニューラルネットワークがあるビット列を予測できた場合、その生成器にニューラルネットワークが利用できる規則性が存在することを示している。

一方、予測精度が50％付近であったとしても、その乱数生成器が暗号学的に安全であることの証明にはならない。

予測できなかった原因として、生成器が予測困難であること以外にも、次の可能性がある。

* 入力長が不足している。
* 学習データ数が不足している。
* ニューラルネットワークの構造が適切でない。
* 学習が十分に収束していない。

また、生成器に非線形処理を加えただけで、暗号学的安全性が保証されるわけではない。

本実験は、特定のニューラルネットワークを用いて乱数列の予測可能性を調べるものであり、暗号学的安全性そのものを証明するものではない。

### 11．使用環境

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib
* Python標準ライブラリの`secrets`
* ノートPCのCPU

### 12．本企画の意義

本企画を通して、ニューラルネットワークが時系列データ中の規則性を学習する過程を確認できる。

また、周期が長く統計的に乱数らしく見えることと、暗号学的に将来の値を予測できないことが異なる性質であると理解できる。

さらに、LFSRが暗号用乱数生成器として単独では不十分である理由、非線形処理の役割、およびニューラルネットワークによる予測実験の限界について考察する。
